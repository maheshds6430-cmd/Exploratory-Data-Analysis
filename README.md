# Exploratory-Data-Analysis
Exploratory Data Analysis Project using Python

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a dataset to understand patterns, relationships, and trends.

## Objectives
- Data cleaning and preprocessing
- Handling missing values
- Outlier detection
- Correlation analysis
- Data visualization

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Identified important features influencing the dataset
- Discovered correlations between variables
- Generated meaningful visual insights

## Files in Repository
- EDA_Project.ipynb (Notebook with full analysis)
- index.html (Rendered HTML version of project)
